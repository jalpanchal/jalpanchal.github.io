Interactive PPG Filtering Game - How to Play

Welcome to the PPG Filtering Game! Your goal is to clean the noisy PPG signal and match it as closely as possible to the clean reference signal. Aim for an RMSE (Root Mean Squared Error) below 0.12 to win!

  • Highpass Filter: Removes low-frequency noise (e.g., baseline drift). Adjust the cutoff frequency to control the minimum frequency that passes through.
  • Lowpass Filter: Removes high-frequency noise (e.g., motion artifacts). Adjust the cutoff frequency to control the maximum frequency that passes through.
  • Rolling Mean Filter: Smooths short-term fluctuations. Adjust the window size to control the level of smoothing.

How to Play:

  1. Start with the noisy signal (gray) and observe how it differs from the clean signal (black).
  2. Apply the filters one by one using the checkboxes.
  3. Adjust the sliders to find the optimal cutoff frequencies and window size.
  4. Watch the RMSE in the plot title - aim to reduce it below 0.12 to turn the filtered signal (blue) green.
  5. Experiment and have fun! There are multiple ways to reach the target RMSE.
